### 2. **المقارنة بناءً على المحتوى (Content Scanning)**

يستخدم النظام عملية متعددة المراحل:

```python name=core/fs.py url=https://github.com/arsenetar/dupeguru/blob/main/core/fs.py
# Partial hashing للملفات الكبيرة
```

**خطوات المقارنة:**
- **First Pass**: مقارنة حجم الملف
- **Second Pass**: مقارنة Hash جزئي (partial hash) للملفات الكبيرة
- **Third Pass**: مقارنة Hash كامل (full hash)
